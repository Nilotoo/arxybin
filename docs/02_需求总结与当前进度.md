## 二、DSP 信号链（已完成）

```
输入 → InputGain → GranularEngine → Distortion → Bitcrusher →
       Stutter → BufferShuffler → Dry/Wet Mix → OutputGain → 输出
```

### 2.1 颗粒合成引擎 (GranularEngine)
- 4 秒环形缓冲区，最多 256 个同时活跃颗粒
- 每个颗粒：位置、长度、音高、声像、方向（正向/反向）、幅度包络（Hann 窗）
- 线性插值读取，支持反馈

### 2.2 扫描模式 (Scan)
- **Forward**：从左到右线性扫描
- **Reverse**：从右到左线性扫描
- **Bidirectional**：来回扫描
- **Random**：随机位置跳跃
- scanSpeed 控制扫描速度

### 2.3 失真 (Distortion)
- **Soft Clip**（tanh 饱和）
- **Hard Clip**（硬削波）
- **Foldback**（折叠失真）
- **Fuzz**（法兹）
- Drive 0-100% + Random 开关

### 2.4 位深/采样率降低 (Bitcrusher)
- 位深 1-24 bits + Random
- 采样率 1x/2x/4x/8x/16x/32x/64x + Random
- 平滑过渡防止噪音

### 2.5 卡顿 (Stutter)
- 概率触发缓冲冻结/循环
- Stutter 长度 1-2000ms
- 概率 + 长度各自有 Random 开关

### 2.6 随机打乱 (BufferShuffler)
- 按段落大小随机重排
- Shuffle Amount 0-100% + Random
- Shuffle Size 1-1000ms + Random

### 2.7 LFO 系统（双 LFO）
- 2 个独立 LFO
- 参数：Rate (0.01–1000Hz)、Depth (0–100%)、Target (Off/Pitch/Position/Pan/Size/Density)、Waveform (Sine/Triangle/Saw/Square/S&H)
- 每采样点计算，实时调制颗粒参数

## 六、关键约束与教训

1. **FL Studio 兼容性**：自定义 AudioProcessorEditor 子类在 FL Studio 中崩溃。GenericAudioProcessorEditor 继承可用。
2. **不能用的**：`setLookAndFeel()`、构造时启动 Timer、`attachToComponent()`
3. **能用的**：Generic 编辑器继承 + 覆盖 overlay（标题/波形/底栏）+ `setColour()` 直接着色
4. **安装方式**：直接复制 `arxybin..vst3` 文件夹到 `C:\Program Files\Common Files\VST3\`
5. **静态链接**：`CMAKE_MSVC_RUNTIME_LIBRARY` 全局设置避免依赖外部 DLL
